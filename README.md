# website-template
Webサイトを作成するためのテンプレート。

## 目的
- Gulp & Sass（SCSS）によるCSSコーディングの効率化
- コーディング、ディレクトリの規約統一

## 用意している機能
- Neatによるグリッドレイアウトシステム
- RWD対応のモジュラースケール（オリジナル）
- 余白の統一によるバーティカルリズムの実現

## 必要環境
- [Node.js](https://nodejs.org)
- [Yarn](https://yarnpkg.com)

## 利用する手順
```
# Node.jsパッケージのインストール
yarn install

# Yarnが無い場合
npm install
```

```
# 開発開始
gulp
```
